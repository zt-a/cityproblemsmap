# Реализованные функции CityProblemMap Frontend

Дата: 2026-04-10

## ✅ Полностью реализованные модули

### 1. **Notifications (Уведомления)** 
**Страница:** `/notifications`

**Функционал:**
- Список всех уведомлений пользователя
- Фильтрация: все / непрочитанные
- Фильтрация по типу уведомления
- Отметка как прочитанное (одно или все)
- Удаление уведомлений (одно или все)
- Статистика: всего / непрочитанные / прочитанные
- Иконки и цветовая индикация по типам

**API эндпоинты:**
- `GET /api/v1/notifications/` - список уведомлений
- `GET /api/v1/notifications/stats` - статистика
- `POST /api/v1/notifications/mark-read` - отметить как прочитанное
- `POST /api/v1/notifications/mark-all-read` - отметить все
- `DELETE /api/v1/notifications/{id}` - удалить уведомление
- `DELETE /api/v1/notifications/` - удалить все

**Типы уведомлений:**
- Изменение статуса проблемы
- Новый комментарий
- Ответ на комментарий
- Проблема решена
- Новый подписчик
- Упоминание
- Новое достижение
- Обновление подписки

---

### 2. **User Settings (Настройки пользователя)**
**Страница:** `/settings`

**Функционал:**
- Настройки способов доставки уведомлений:
  - Email уведомления (вкл/выкл)
  - Push уведомления (вкл/выкл)
- Настройки типов уведомлений (8 типов):
  - Изменение статуса проблемы
  - Новый комментарий
  - Ответ на комментарий
  - Проблема решена
  - Новый подписчик
  - Упоминание
  - Новое достижение
  - Обновление подписки
- Сохранение настроек с подтверждением

**API эндпоинты:**
- `GET /api/v1/settings/notifications` - получить настройки
- `PATCH /api/v1/settings/notifications` - обновить настройки

---

### 3. **Gamification (Геймификация)**
**Страница:** `/achievements`

**Функционал:**
- Статистика пользователя:
  - Уровень и опыт (с прогресс-баром)
  - Количество достижений
  - Рейтинг в общем зачете
  - Общий прогресс
- Список всех достижений с фильтрами:
  - Все
  - Получено
  - Заблокировано
- Карточки достижений с:
  - Иконкой и названием
  - Редкостью (обычное/редкое/эпическое/легендарное)
  - Описанием
  - Прогресс-баром (для незавершенных)
  - Датой получения
  - Наградой (опыт)

**API эндпоинты:**
- `GET /api/v1/gamification/stats` - статистика пользователя
- `GET /api/v1/gamification/achievements` - список достижений
- `POST /api/v1/gamification/check-achievements` - проверить новые

---

### 4. **Subscriptions (Подписки)**
**Страница:** `/subscriptions`

**Функционал:**
- Список всех подписок пользователя
- Группировка по типам:
  - Проблемы
  - Зоны (районы)
  - Пользователи (будущее)
- Статистика по типам
- Фильтрация по типу
- Отображение настроек уведомлений для каждой подписки:
  - Изменение статуса
  - Комментарии
  - Решение
- Отписка от проблем/зон
- Переход к объекту подписки

**API эндпоинты:**
- `GET /api/v1/subscriptions/` - список подписок
- `POST /api/v1/subscriptions/problems/{id}` - подписаться на проблему
- `DELETE /api/v1/subscriptions/problems/{id}` - отписаться от проблемы
- `POST /api/v1/subscriptions/zones/{id}` - подписаться на зону
- `DELETE /api/v1/subscriptions/zones/{id}` - отписаться от зоны
- `PATCH /api/v1/subscriptions/{id}` - обновить настройки подписки

---

### 5. **Social Feed (Лента активности)**
**Страница:** `/feed`

**Функционал:**
- Лента активности подписок
- Отображение действий пользователей:
  - Создание проблемы
  - Решение проблемы
  - Добавление комментария
  - Подписка на пользователя
  - Получение достижения
- Аватары пользователей
- Временные метки (относительные)
- Переход к связанным объектам

**API эндпоинты:**
- `GET /api/v1/social/feed` - лента активности
- `GET /api/v1/social/profile/{user_id}` - профиль пользователя
- `POST /api/v1/social/follow/{user_id}` - подписаться
- `DELETE /api/v1/social/follow/{user_id}` - отписаться
